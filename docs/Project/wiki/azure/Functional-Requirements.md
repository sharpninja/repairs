# Functional Requirements (MCP Server)

## FR-A11Y-001 WCAG 2.2 AA accessibility conformance

The AI Auto Repairman PWA, admin dashboard, and guide slideshow must conform to WCAG 2.2 Level AA so keyboard-only and assistive-technology users can complete every core task (browse/install guides, submit reviews, follow safety checklists, use the AI chat).
Scope: layer-1+
